Resumo do sistema

É um sistema web, onde o administrador do serviço pode inserir, excluir ou atualizar cardápios no sistema.

Os cardápios serão visíveis para todos os usuários e permite que o mesmo avalie o serviço do restaurante.
	- Comentar cardápios
	- Nível de satisfatibilidade

É possível visualizar estatísticas sobre as avaliações realizadas.
